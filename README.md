# 🏥 AI Hospital Resource Allocation System

## 📌 Problem

Hospitals often face critical shortages of resources such as beds, oxygen, and staff due to the lack of real-time demand prediction and intelligent allocation systems.

## 💡 Solution

This project presents an AI-powered system that predicts patient inflow in real time and provides smart suggestions for efficient resource allocation within hospitals.

## 🚀 Features

* 📊 Real-time patient inflow prediction using AI
* 🏥 Monitoring of hospital resources (beds, oxygen, staff)
* ⚠️ Intelligent alerts for potential shortages
* 🤖 Decision support for optimal resource allocation

## 🛠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas, NumPy
